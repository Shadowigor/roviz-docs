QVariant SettingsScopePrivate::value ( const QString &  key,
const SettingsScope *  scope,
bool  searchAll = true 
) const

Returns the value associated with key or an invalid variant, if no setting for key was found.

If key cannot be found in this scope and searchAll is true, all parent scopes up the parent chain are consulted until a defined value was found.

If searchAll is false, only scope is consulted.
